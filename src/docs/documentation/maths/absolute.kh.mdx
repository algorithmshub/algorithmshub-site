# Absolute

**តើតម្លៃ Absolute ជាអ្វី?**

តម្លៃ absolute គឺតម្លៃមិនអវិជ្ជមាន។

**ឧទាហរណ៍**

សរសេរ function មួយដែល return ចំនួនលេខនៃតម្លៃ absolute ។

**Java**

```js
public class Absolute {
  public static void main(String[] args) {
    System.out.println(absVal(-0) == Math.abs(0));
    System.out.println(absVal(1) == Math.abs(1));
    System.out.println(absVal(-2) == Math.abs(2));
  }

  /**
   * ប្រសិនបើតម្លៃតូចជាងសូន្យធ្វើឱ្យតម្លៃវិជ្ជមាន
   *
   * @param value a number
   * @return the absolute value of a number
   */
  public static int absVal(int value) {
    return value < 0 ? -value : value;
  }
}
```
